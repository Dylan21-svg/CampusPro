**CampusPro** is a high-performance school management platform featuring a premium SaaS dashboard aesthetic, role-based access control, and advanced institutional analytics.

## ✨ Key Features

- 🌓 **Adaptive UI**: Seamless Light/Dark mode transitions.
- 📱 **Fully Responsive**: Optimized for Mobile, Tablet, and Desktop.
- 🔐 **Secure Auth**: PBKDF2 password hashing and granular RBAC.
- 📊 **Pro Analytics**: Real-time data visualization with Chart.js.
- 💰 **Payment Integration**: Support for Stripe, PayPal, and Mobile Money (MTN/Orange).
- 📂 **Management Suites**: Blueprinted modules for Students, Teachers, Classes, and Subjects.

## 🚀 Quick Start

### 1. Requirements

Ensure you have Python 3.8+ installed.

### 2. Installation

```bash
# Clone the repository
git clone https://github.com/Dylan-svg/CampusPro.git
cd "School Management System"

# Install dependencies
pip install -r requirements.txt
```

### 3. Execution

```bash
python app.py
```

_The system automatically initializes the database on first run._
